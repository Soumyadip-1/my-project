This project is a modern, responsive Admin Dashboard designed using HTML and CSS.
It includes a fixed header with a title and a functional Dark Mode toggle, a sidebar for navigation (Home, Users, Settings)
a main content section displaying a welcome message, and a footer showing the copyright.
The layout is structured using Flexbox for a clean and adaptive design that works well on various screen sizes.
The Dark Mode feature allows users to switch between light and dark themes for better accessibility and user comfort.
This dashboard can be extended to manage users, view analytics, or control system settings.


Final Output
The resulting page is a simple, professional-looking admin dashboard with:
A fixed layout,
Responsive content structure,
And a Dark Mode toggle to enhance user experience.
